# Othello
Othello is a strategy board game for two players (user and PC), played on an 8Ã—8 uncheckered board.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

Things you need to install if you already haven't:

```
sudo apt-get install indent ctagas libncurses5-dev libncursesw5-de

opencv
```
